# STM32 两轮自平衡小车学习项目

## 项目介绍
这是我自学嵌入式开发时完成的STM32两轮自平衡小车学习项目。
项目核心基于PID算法实现小车的直立平衡控制，同时涵盖了MPU6050姿态传感器驱动、电机驱动、串口通信等嵌入式开发的核心知识点，全程从硬件电路设计到代码编写均为自主学习复刻完成，仅用于个人学习与技术积累，非商用项目，非自创。

## 仓库内容说明
- car/car_PID：平衡小车核心源码，包含PID算法实现、MPU6050驱动、电机控制等自主编写的学习代码
- Self_balancing_car：个人优化调试版源码，在基础版上做了参数优化与功能完善
- Hardware：自主绘制的平衡小车硬件PCB工程文件（嘉立创EDA格式），包含原理图与PCB版图
- 配套PDF文档：平衡小车开发手册、PID控制原理、运动学模型等学习参考资料

## 补充说明
本项目仅为个人自学嵌入式开发的练习作品，所有代码与硬件设计均用于学习交流，无任何商用意图。
项目涉及的PID算法、传感器驱动、硬件电路等内容，均为学习过程中的实践成果，适合嵌入式入门学习者参考。
